//==================================================================
// 操作方法
//==================================================================
 [W] 上移動
 [S] 下移動
 [SPACE] 弾発射
 [マウス左クリック] タイトル、リザルトのボタンを押す
 [ESC] 実行ファイルを閉じる

//==================================================================
// ゲームのルール
//==================================================================
右側にいるブロックに囲まれた大きなコアを持つボスを倒すとクリアです。
途中で雑魚敵が出現するため、弾を当ててブロックを壊していきます。
雑魚敵もコアがあり、壊すと一撃で倒せます。
雑魚敵がプレイヤーを通り過ぎるとボスが近づいてきて体力が減ります。
体力が0になる、弾がなくなるとゲームオーバーです。

//==================================================================
// アピールポイント
//==================================================================
敵のパラメータを自由にインスペクタ上で調整できるようにしたので一つのスクリプトで
複数の種類の敵を作成できるようになっています。
敵のブロック管理をリストで管理して、スコアと連携できるようにしました。
またボスの行動をクラス分けして簡単に調整ができるようにしています。
