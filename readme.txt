//==================================================================
// ゲームのルール
//==================================================================
制限時間内にゴールを目指して先に進む作品です。
パルクールを駆使したアクションで壁を蹴り走り、敵でさえ軽やかに倒していくアクションゲームになります。
ボスを倒しゴールまでの足場を出すことでクリアへ進めます。

//==================================================================
// アピールポイント
//==================================================================
階層構造を複数使ったモデル管理を行っているところです。
ボスのパーツは上半身、下半身、左右の大きな腕とすべて独立した動きを連動させています。
これらはマトリックスを用いた階層構造の応用で管理して動作させています。
